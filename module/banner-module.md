### Overview

Use the Banner module to add a focused call-to-action section anywhere on the page. It works well for short campaigns, announcements, sales prompts, or strong single-message blocks that need to stand apart from surrounding content.

You can pair concise copy with a button and optional background treatment, then use the settings to control spacing, alignment, and how the banner behaves visually. It is designed for moments where you want visitors to stop, read, and act.

### Content Fields

**Heading**
The main title shown at the top of the Banner module.

**Intro**
Optional short line or subheading under the heading.

**Button**
Optional link or call to action for this item or section.

**Link Type**
Optional link or call to action for this item or section. Options include <code>Module</code>, <code>Email</code>, <code>Website</code>, <code>File</code>.

**Email Link**
Optional link or call to action for this item or section. Enter a valid email address.

**Module Link**
Optional link or call to action for this item or section. Options include <code>hero</code>, <code>About</code>, <code>Services</code>, <code>Team</code>, <code>Gallery</code>, <code>Partners</code>, <code>review</code>, <code>counter</code>, <code>Contact</code>, <code>Banner</code>. Enter a valid email address.

**Website Link**
Optional link or call to action for this item or section. Enter the website address. Please include the https://.

**File**
Sets the file value for this call-to-action banner.

**Window**
Choose one of the available options: <code>Open in same Tab</code>, <code>Open in new Tab</code>.

### Settings Fields

**Active**
Control whether the Banner module should display for visitors. Set it to <code>No</code> when the section is not in use or not ready. Options include <code>Yes</code>, <code>No</code>.

**Order**
A number that controls where this module appears among other homepage modules (lower numbers typically appear higher on the page).

**Menu**
Optional label in the navigation with jump links that scroll to this module.

**Text Colour**
Sets the text colour value for this call-to-action banner.

**Overlay**
Sets the overlay value for this call-to-action banner.

**Parallax**
Choose one of the available options: <code>Yes</code>, <code>No</code>.
