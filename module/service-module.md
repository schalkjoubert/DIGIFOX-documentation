Use the Services module to explain what you offer in a clear set of cards or blocks. It works well for service lists, capabilities, packages, or any area where visitors need a quick overview of how you can help.

Each service can include a title, description, image or icon where supported, and optional action. Settings control layout, columns, spacing, and styling so the section can be concise or more detailed.

### Services | Content

**Heading**
The main title shown at the top of the Services module.

**Intro**
Optional short line or subheading under the heading.

**Description**
Optional longer text for this section. Use it when visitors need more context than the intro provides.

**Service**
The list of Services items visitors see. Use <code>Add Service Card</code> for each item.

**Title**
The primary title for this item.

**Subtitle**
The primary title for this item.

**Description**
Optional longer text for this section. Use it when visitors need more context than the intro provides.

**Button**
Optional link or call to action for this item or section.

**Link Type**
Optional link or call to action for this item or section. Options include <code>Module</code>, <code>Email</code>, <code>Website</code>, <code>File</code>.

**Email Link**
Optional link or call to action for this item or section. Enter a valid email address.

**Module Link**
Optional link or call to action for this item or section. Options include <code>hero</code>, <code>About</code>, <code>Services</code>, <code>Team</code>, <code>Gallery</code>, <code>Carousel3</code>, <code>review</code>, <code>counter</code>, <code>Contact</code>, <code>Banner</code>. Inactive modules (greyed out) cannot be selected. If a module you had selected is later turned off, the link will default to Contact.

**Website Link**
Optional link or call to action for this item or section. Enter the website address. Please include the https://.

**Window**
Enter the website address. Please include the https:// Options include <code>Open in same Tab</code>, <code>Open in new Tab</code>.

**File**
Enter the website address. Please include the https://

**Image or Icon**
Optional image used by this item or section. Use consistent image sizes where possible for a cleaner layout. Options include <code>Icon</code>, <code>Image</code>, <code>None</code>.

**Icon**
Optional icon used to visually support this item.

**Image**
Optional image used by this item or section. Use consistent image sizes where possible for a cleaner layout. Recommended Size: 600px by 450px.

### Services | Settings

**Active**
Control whether the Services module should display for visitors. Set it to <code>No</code> when the section is not in use or not ready. Options include <code>Yes</code>, <code>No</code>.

**Order**
A number that controls where this module appears among other homepage modules (lower numbers typically appear higher on the page).

**Menu**
Optional label in the navigation with jump links that scroll to this module.

**Display**
Choose how items are shown. Grid-style options show items together, while slider-style options let visitors move through them.

**Columns**
How many columns the layout uses on wider screens.

**Gap**
Space between items in pixels.

**Padding**
Inner spacing for the block, in pixels, so content does not sit flush against the edges.

**Image Radius**
Rounds the corners of images, as a percentage. Use <code>-1</code> where available to use the global image radius from <code>Setup</code>.

**Image Crop**
Control how images are cropped so items can display with a consistent shape. Options include <code>Natural</code>, <code>4:3</code>, <code>1:1</code>.

**Layout**
Control the overall layout or media orientation for this module. Options include <code>Portrait:</code>, <code>Landscape</code>.

**Text Align**
Aligns the text within this module. Options include <code>Left</code>, <code>Center</code>.

**Border**
Adds a visible border when enabled. The color is set in <code>Setup</code>. Options include <code>Yes</code>, <code>No</code>.

**Background**
Applies the theme background treatment when enabled. The color is set in <code>Setup</code>. Options include <code>Yes</code>, <code>No</code>.

**Equal Heights**
Keeps items at the height of the tallest item so rows feel balanced. Options include <code>Yes</code>, <code>No</code>.

**Pagination**
Navigation options for how visitors move between slider items. Options include <code>Dots</code>, <code>Arrows</code>, <code>None</code>.

**Speed**
Set the transition time between slides, in milliseconds.

**Delay**
When using a slider, set the pause between automatic advances in milliseconds. Use <code>0</code> to disable autoplay.

**Custom Style**
Advanced option: set to <code>Yes</code> to unlock the extra CSS field for custom style tweaks. Options include <code>Yes</code>, <code>No</code>.

**Custom CSS**
Optional CSS for fine-tuning this module. Best used by designers or Digifox support.
