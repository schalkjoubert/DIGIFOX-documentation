### Overview

Use the Subscribe module to invite visitors into a mailing list, enquiry flow, or lead capture moment. It creates a focused section for signup copy and form placement.

The module is useful for newsletter prompts, download offers, campaign signups, or simple subscriber forms. Settings control layout and presentation so the form feels like part of the page rather than an external widget.

### Content Fields

**Heading**
The main title shown at the top of the Subscribe module.

**Intro**
Optional short line or subheading under the heading.

### Settings Fields

**Active**
Control whether the Subscribe module should display for visitors. Set it to <code>No</code> when the section is not in use or not ready. Options include <code>Yes</code>, <code>No</code>.

**Order**
A number that controls where this module appears among other homepage modules (lower numbers typically appear higher on the page).

**Menu**
Optional label in the navigation with jump links that scroll to this module.
