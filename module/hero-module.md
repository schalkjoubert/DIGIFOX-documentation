### Overview

Use the Hero module for the main opening statement of a page. It is designed for strong headings, supporting copy, imagery or video, and clear calls to action that set the tone for the rest of the page.

The module gives you control over the headline area, media, buttons, and layout so the first screen can be direct, branded, and conversion-focused. Settings help adapt the hero to different page goals and screen sizes.

### Content Fields

**Hero Background**
Choose one of the available options: <code>Background Image/s</code>, <code>Background Video</code>, <code>Background Colour</code>.

**Video**
Paste an embeddable video or media link, such as YouTube or Vimeo where supported.

**Image/s**
Optional image used by this item or section. Use consistent image sizes where possible for a cleaner layout.

**Heading**
The main title shown at the top of the Hero module.

**Intro**
Optional short line or subheading under the heading.

**Footer**
An optional short intro, displayed below the heading.

**Button**
Optional link or call to action for this item or section.

**Link Type**
Optional link or call to action for this item or section. Options include <code>Module</code>, <code>Email</code>, <code>Website</code>, <code>File</code>.

**Email Link**
Optional link or call to action for this item or section. Enter a valid email address.

**Module Link**
Optional link or call to action for this item or section. Options include <code>hero</code>, <code>About</code>, <code>Services</code>, <code>Team</code>, <code>Gallery</code>, <code>Carousel3</code>, <code>review</code>, <code>counter</code>, <code>Contact</code>, <code>slider</code>, <code>Banner</code>, <code>Embed</code>, <code>Subscribe</code>. Inactive modules (greyed out) cannot be selected. If a module you had selected is later turned off, the link will default to Contact.

**Website Link**
Optional link or call to action for this item or section. Enter the website address. Please include the https://.

**File**
Sets the file value for this hero section.

**Window**
Choose one of the available options: <code>Open in same Tab</code>, <code>Open in new Tab</code>, <code>Force Download</code>.

### Settings Fields

**Active**
Control whether the Hero module should display for visitors. Set it to <code>No</code> when the section is not in use or not ready. Options include <code>Yes</code>, <code>No</code>.

**Order**
A number that controls where this module appears among other homepage modules (lower numbers typically appear higher on the page).

**Menu**
Optional label in the navigation with jump links that scroll to this module.

**Text Colour**
Sets the text colour value for this hero section.

**Overlay Colour**
Sets the overlay colour value for this hero section.

**Overlay Area**
Choose one of the available options: <code>None</code>, <code>Full</code>, <code>Text</code>.

**Height**
Sets the height value for this hero section.

**Custom Style**
Advanced option: set to <code>Yes</code> to unlock the extra CSS field for custom style tweaks. Options include <code>Yes</code>, <code>No</code>.

**Custom CSS**
Optional CSS for fine-tuning this module. Best used by designers or Digifox support.
