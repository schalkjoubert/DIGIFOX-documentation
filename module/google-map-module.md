### Overview

Use the Google Map module to add a visual location reference to the Contact area. It is useful when visitors need to find a physical office, store, showroom, venue, or service area without leaving the page.

The map is managed as part of the Contact module, but it can be documented and displayed as its own module on the Digifox website. Enable the map, paste the Google Maps embed iframe, and the Contact section will show the map alongside your other contact details.

### Content Fields

**Google Map**
Controls whether the map should display in the Contact section. Set this to Yes when you want to show an embedded Google Map. Options include <code>Yes</code>, <code>No</code>.

**Iframe**
Paste the embed iframe code copied from Google Maps. In Google Maps, search for the location, choose Share, select Embed a map, then copy and paste the iframe code into this field.

### Settings Fields

**Contact Module Settings**
The Google Map uses the Contact module's display settings. Use the Contact module to control whether the section is active, where it appears on the page, and any custom CSS needed for final layout adjustments.
