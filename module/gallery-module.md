### Overview

Use the Gallery module to present a collection of images in a clear visual layout. It works well for portfolios, venues, product details, events, before-and-after examples, or any page where imagery carries the message.

You can add multiple images and control how they crop, space, and align. The settings help keep the gallery consistent even when source images have different shapes or sizes.

### Content Fields

**Heading**
The main title shown at the top of the Gallery module.

**Introduction**
Optional short supporting line under the heading. An optional short intro, displayed below the heading.

**Images**
Optional image used by this item or section. Use consistent image sizes where possible for a cleaner layout.

### Settings Fields

**Active**
Control whether the Gallery module should display for visitors. Set it to <code>No</code> when the section is not in use or not ready. Options include <code>Yes</code>, <code>No</code>.

**Order**
A number that controls where this module appears among other homepage modules (lower numbers typically appear higher on the page).

**Menu**
Optional label in the navigation with jump links that scroll to this module.

**Gap**
Space between items in pixels.

**Border**
Adds a visible border when enabled. The color is set in <code>Setup</code>. Options include <code>Yes</code>, <code>No</code>.

**Background**
Applies the theme background treatment when enabled. The color is set in <code>Setup</code>. Options include <code>Yes</code>, <code>No</code>.

**Custom Style**
Advanced option: set to <code>Yes</code> to unlock the extra CSS field for custom style tweaks. Options include <code>Yes</code>, <code>No</code>.

**Custom CSS**
Optional CSS for fine-tuning this module. Best used by designers or Digifox support.
