### Overview

Use the Slideshow module to rotate through visual slides, messages, or promotional panels. It is useful when you need several hero-like moments in one area without making the page too long.

Slides can combine images and copy where supported, while settings control movement, timing, pagination, sizing, and display behaviour. This gives you a flexible way to present multiple featured messages.

### Settings Fields

**Active**
Control whether the Slideshow module should display for visitors. Set it to <code>No</code> when the section is not in use or not ready. Options include <code>Yes</code>, <code>No</code>.

**Order**
A number that controls where this module appears among other homepage modules (lower numbers typically appear higher on the page).

**Menu**
Optional label in the navigation with jump links that scroll to this module.

**Transition**
Choose one of the available options: <code>Fade</code>, <code>Slide</code>.

**Image Crop**
Control how images are cropped so items can display with a consistent shape. Options include <code>Natural</code>, <code>4:3</code>, <code>16:9</code>.

**Pagination**
Navigation options for how visitors move between slider items. Options include <code>Dots</code>, <code>Arrows</code>, <code>None</code>.

**Speed**
Set the transition time between slides, in milliseconds.

**Delay**
When using a slider, set the pause between automatic advances in milliseconds. Use <code>0</code> to disable autoplay.

**Text Colour**
Sets the text colour value for this slideshow.

**Overlay**
Sets the overlay value for this slideshow.

**Height**
Sets the height value for this slideshow.

**Custom Style**
Advanced option: set to <code>Yes</code> to unlock the extra CSS field for custom style tweaks. Options include <code>Yes</code>, <code>No</code>.

**Custom CSS**
Optional CSS for fine-tuning this module. Best used by designers or Digifox support.
